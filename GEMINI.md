# Arrows: Engineering Diagramming Tool

Arrows is a hybrid diagramming tool designed to bridge the gap between **Diagram-as-Code** (like Mermaid) and **High-Fidelity WYSIWYG Editors** (like draw.io). It is specifically tailored for software engineering and architectural diagrams that need to be both readable by LLMs and presentation-ready for humans.

## Core Concepts

### 1. Dual-State Textual Representation
Arrows uses a single text block that contains two distinct parts:
- **Semantic Graph (Mermaid):** A standard Mermaid flowchart block. This is what an LLM generates and what standard Markdown viewers render.
- **Visual State (Arrows Config):** A hidden metadata block (using Mermaid comments) that stores high-fidelity layout data, icons, and colors.

### 2. LLM-First Workflow
Because the primary format is standard Mermaid, LLMs can effortlessly:
- Generate initial architectural sketches.
- Reason about data flows and service dependencies.
- Update the diagram's logic without breaking the visual styling.

### 3. Bi-Directional Synchronization
The tool maintains a "Single Source of Truth" in the text editor:
- **Text -> Canvas:** Editing the Mermaid syntax immediately updates the nodes and edges on the canvas.
- **Canvas -> Text:** Dragging nodes, changing colors, or selecting icons updates the hidden configuration block in the text.

### 4. Smart Auto-Layout
If a diagram is pasted without visual metadata (e.g., newly generated by an LLM), Arrows employs the **Dagre** layout engine to automatically arrange the components into a clean, hierarchical structure.

## Data Format

```mermaid
flowchart TD
  WebApp[Web Application]
  DB[(PostgreSQL Database)]
  
  WebApp --> DB

%% --- arrows-config --- %%
{
  "nodes": {
    "WebApp": { "x": 100, "y": 150, "color": "#3b82f6", "icon": "Server" },
    "DB": { "x": 400, "y": 150, "color": "#10b981", "icon": "Database" }
  },
  "edges": {}
}
```

## Technical Stack
- **React Flow:** Powering the interactive node-based canvas.
- **Dagre:** Providing automatic hierarchical layout.
- **Lucide React:** Providing a professional set of engineering icons.
- **Vite + TypeScript:** Ensuring a fast and type-safe development experience.
